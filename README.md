# AI-Project-Manager-Portfolio-Generator
An AI-powered application that automatically analyzes GitHub repositories and generates professional documentation and portfolio content.

---

## 📌 Overview

This application analyzes GitHub repositories and generates:

* 📄 Professional README files
* 💼 LinkedIn-ready project summaries
* 📊 Insights from commits and issues

Built for **students and small dev teams** who want to showcase their work without spending hours on documentation.

## ⚙️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** FastAPI
* **AI:** Gemini Flash / Llama
* **Database:** SQLite
* **Other:** Docker, GitHub API

---

## 🧠 How It Works

1. User submits a GitHub repository URL
2. Backend fetches repository data
3. AI analyzes development history
4. Results are stored and processed
5. UI displays insights + generated content
